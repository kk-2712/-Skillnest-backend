# SkillNest – Full Stack Course Platform

## Tech Stack
- React
- Node.js + Express
- MongoDB
- JWT Authentication

## Features
- User authentication
- Course listing
- Enrollment system
- Smart course recommendations
- Responsive UI

## How to Run
### Backend
cd Skillnest-backend
npm install
npm run dev

### Frontend
cd client
npm install
npm start
